# Spotify Music Analysis

Simple exploratory data analysis of a Spotify dataset to understand how audio features relate to song popularity and genres.

## Tools
- Python
- Pandas
- Matplotlib
- Seaborn

## What’s included
- Data cleaning (duplicates, missing values)
- Popularity distribution
- Top genres analysis
- Audio feature distributions
- Correlation heatmap
- Genre comparison (energy, danceability, popularity)

##  Goal
Explore how features like energy and danceability vary across genres and relate to popularity.

pip install pandas matplotlib seaborn
